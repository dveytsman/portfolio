Trellow

Project Management Applications

A Full Stack Trello clone that allows for the creation of boards lists and cards. Allows users to keep track of their tasks. Utilizes a Rails back-end with a React and Redux front-end with a jBuilder data retrieval. Fully functional user authentication pattern. CRUD (Create, Read, Update, and Delete) pattern for Boards, Lists and Cards.

  • Frontend: React
  • Backend: Rails

Sharp Swiper

Fruit Ninja Remake

Fully interactive swiping game. Hours of fun sure to keep you hooked and trying to beat your high score. Game uses Vanilla Javascript, HTML, and CSS to create a reactive gameplay experience. Difficulty increases steadily as game progresses. The game uses spritesheets to animate the asteroids and the explosions.

Minique

Light weight DOM manipulation library.

Minique is a JavaScript library that allows for the utilization of the DOM manipulation. It allows the user to easily select and modify elements in the DOM quickly. Created to mimic the functionality of the jQuery library but with minimal wasted space.

  • Language: Javascript
